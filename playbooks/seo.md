# SEO Playbook

> Standard operating procedure for any SEO engagement.

## Phase 1: Discovery (Week 1)

- [ ] Run technical audit (crawl for broken links, slow pages, missing meta)
- [ ] Pull 12 months of GSC data: impressions, clicks, CTR, position
- [ ] Pull 12 months of GA data: traffic by page, conversions, bounce rate
- [ ] Run keyword gap analysis vs top 3 competitors
- [ ] Run backlink gap analysis vs top 3 competitors
- [ ] Identify keyword clusters (informational / commercial / transactional)
- [ ] Document current state in `clients/<slug>/analytics/seo-audit.md`

## Phase 2: Strategy (Week 2)

- [ ] Pick 5 priority pages for quick wins (positions 8-20, high impressions)
- [ ] Pick 10 long-tail keywords to build content around
- [ ] Pick 3 link-building targets (industry publications, podcasts, partners)
- [ ] Set 90-day KPI targets (conservative / realistic / stretch)
- [ ] Get client approval on the strategy doc

## Phase 3: On-Page Optimization (Weeks 2-4)

For each priority page:
- [ ] Rewrite title tag (60 chars, primary keyword near front)
- [ ] Rewrite meta description (155 chars, includes keyword + CTA)
- [ ] Add H1 with primary keyword (only one H1 per page)
- [ ] Restructure H2s with secondary keywords
- [ ] Add internal links (3-5 contextual links to related pages)
- [ ] Add structured data (Article, Product, FAQ as appropriate)
- [ ] Compress images, add alt text
- [ ] Check Core Web Vitals

## Phase 4: Content Production (Ongoing)

- [ ] Publish 4 blog posts/month, targeting 1 primary + 3-5 secondary keywords each
- [ ] Update top 10 pages quarterly (refresh stats, expand sections)
- [ ] Build topical authority: cluster pages around pillar topics

## Phase 5: Link Building (Ongoing)

- [ ] HARO / Qwoted responses (1/week)
- [ ] Guest posts on industry publications (2/month)
- [ ] Resource page link building (10 outreach emails/week)
- [ ] Digital PR: pitch data-driven stories to journalists

## Phase 6: Reporting (Monthly)

Monthly report covers:
- Traffic (organic sessions, MoM, YoY)
- Rankings (movement on target keywords)
- Conversions (form fills, signups, sales from organic)
- Backlinks acquired
- Content published
- Next month priorities

## Tools

| Need | Free | Paid |
|------|------|------|
| Keyword research | Google Keyword Planner, AnswerThePublic | Ahrefs, Semrush |
| Technical audit | Screaming Frog (500 URLs free), PageSpeed Insights | Sitebulb |
| Rank tracking | GSC | Ahrefs, Semrush |
| Backlinks | — | Ahrefs, Semrush |
| Content optimization | — | Surfer SEO, Frase |

## Anti-patterns to avoid

- ❌ Keyword stuffing
- ❌ Buying links
- ❌ Publishing thin content (300 words, no value)
- ❌ Ignoring search intent (ranking for "best CRM" with a landing page)
- ❌ Set-and-forget (SEO is a 6-12 month game)
