# Content Playbook

> How to write content for Farber.Inc clients — blogs, social, email, ads.

## Universal rules (apply to every piece)

1. **Lead with the reader's problem.** Not "We built X." Start with the pain.
2. **Specifics > generalities.** Numbers, names, examples beat abstract claims.
3. **One idea per paragraph.** If a paragraph needs two ideas, split it.
4. **Cut filler.** "It's important to note that…" → delete. "In today's fast-paced world…" → delete.
5. **Active voice.** "We built X" not "X was built by us."
6. **Show, don't tell.** Instead of "our product is reliable" → describe a scenario where it handled the edge case.
7. **Cite sources.** External claims need a link. Statistics need a primary source.

## Brand voice calibration

Every client has a brand voice in `clients/<slug>/brand/voice.md`. Before writing *anything*:
- [ ] Read the voice doc
- [ ] Read 3 recent pieces the client published and approved
- [ ] Match that register, vocabulary, sentence length

If no voice doc exists, ask these questions:
- Are we formal (law, finance) or casual (consumer, lifestyle)?
- Do we use first-person ("I", "we") or third-person?
- Do we use industry jargon or plain language?
- Do we have a recurring tagline or catchphrase?
- Do we use humor? Sarcasm? Empathy? Authority?

## Blog post structure (long-form, 1500-2500 words)

```
Title (60 chars max, primary keyword near front)
Meta description (155 chars, includes keyword + value prop + CTA)

H1 — Restates promise of title

Intro (150-200 words)
- Hook (stat, story, contrarian take)
- Problem statement
- Promise of post ("By the end, you'll know X")
- Roadmap ("We'll cover A, B, C")

H2 — Section 1 (300-500 words)
  H3 — subpoint
  H3 — subpoint

H2 — Section 2 (300-500 words)
  H3 — subpoint
  H3 — subpoint

H2 — Section 3 (300-500 words)
  H3 — subpoint
  H3 — subpoint

H2 — Common mistakes (200-300 words)
  bullet list

H2 — Conclusion (150-200 words)
  - Summary
  - Key takeaway
  - CTA

FAQ (200-300 words, 4-6 questions)
  — Optimized for AEO (see aeo.md)
```

## Social post patterns

### LinkedIn
- 1200-1500 chars sweet spot
- Hook in first line (the "see more" cutoff)
- Line breaks every 1-2 sentences
- End with question or clear CTA
- No external links in post (kills reach) — link in first comment

### Twitter/X
- Single tweet: punchy, 1 idea, end strong
- Thread: 5-8 tweets, each stands alone, last tweet has CTA
- Best performing hooks: numbers, contrarian, "I was wrong", "Here's why"

### Instagram
- Carousel: 7-10 slides, first slide = hook, last = CTA
- Caption: 100-500 chars, line breaks, 3-5 hashtags max in caption (more in first comment)
- Reels: 7-15 seconds, captioned, trending audio

### TikTok
- Hook in first 1.5 seconds
- 15-30 seconds
- Captions burned in (many watch muted)
- Trending sound helps distribution

## Email sequence patterns

### Welcome series (5 emails over 10 days)
1. **Day 0:** Deliver lead magnet + quick win
2. **Day 2:** Story behind the brand (the "why")
3. **Day 4:** Customer success story
4. **Day 6:** Common mistake your audience makes
5. **Day 10:** Soft pitch for core offer

### Nurture (weekly)
- 1 value email, 1 case study, alternating
- Always one CTA per email

## Quality checklist before publishing

- [ ] Matches brand voice
- [ ] Specifics (numbers, names, examples)
- [ ] No filler phrases
- [ ] Active voice throughout
- [ ] Sources cited for external claims
- [ ] Title and meta optimized (if blog)
- [ ] Internal links (3-5 for blogs)
- [ ] CTA present and clear
- [ ] Spell-check + read aloud test
- [ ] Mobile preview checked (if email/social)
